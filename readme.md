**Plataformas online para ensino de tecnologia para crianças**<br>
* Jogos que ensinam programação <br>
  * <a href="https://code.org/">Code.org</a>: Atividades do 1º ano do fundamental ao 3º do médio.<br>
  * <a href="https://game.rodocodo.com/hour-of-code/?_gl=1*1stvi7h*_ga*MTM3MTQwMDI4My4xNzYzNjYyMzU0*_ga_2SJQ2SJ602*czE3NjM2NjIzNTQkbzEkZzAkdDE3NjM2NjIzNTQkajYwJGwwJGgxNTQzNTQ3MjIz">Robo Codo</a>: Dividido entre idades de 4-7 anos e 8-11 anos<br>
  * <a href="https://runmarco.com/">Run, Marco!</a>: Indicado para idades entre 7-12 anos (crianças alfabetizadas).<br>
  * <a href="https://blockly.games/">Blockly Games</a>: Atividades para crianças alfabetizadas (8-12 anos)<br>
  * <a href="https://www.google.com/logos/2017/logo17/logo17.html?hl=pt-BR">Google Doodle para programação</a>: Indicado para crianças não alfabetizadas. Contempla até loops.<br>
  * <a href="https://codecombat.com/">Code Combat</a>: Depois do cadastro de professor e de turma, ensine Pyhton, Javascript, C++ ou Java para crianças de 7 a 12 anos.<br>
  * <a href="https://silentteacher.toxicode.fr/">Silent Teacher Javascript</a>: Ensine Javascript para alunos a partir de 12 anos.<br>
  * <a href="https://silentteacher.toxicode.fr/hour_of_code.html?theme=basic_python">Silent Teacher Python</a>: Ensine Python para crianças a partir de 12 anos.<br>
  * <a href="https://compute-it.toxicode.fr/">Compute it!</a>: Ensine com pseudocódigo com foco em vários desafios. +12 anos. <br>
  * <a href="https://www.education.com/game/cody/?_gl=1*yysebk*_up*MQ..&gclid=CjwKCAjwi-DBBhA5EiwAXOHsGUpwvK19i2WcXTBCsoTDnzX91WMVFBVrEm5qdWaqbRi3kNdzchjApxoCbXIQAvD_BwE">Coding with roly</a>: Após um cadastro, insira seus alunos e ensine programação a partir do ensino infantil (5 anos)<br>
  * [Kid Loco Coding](https://www.kidlocoding.com/basic-coding/): Programação para crianças ainda não alfabetizadas<br>
<br>

* Informática <br>
  * <a href="https://www.typing.com/student/lessons">Typing</a>: Digitação com lições programadas pelo próprio site<br>
  * <a href="https://www.typingclub.com/">Typing Club</a>: Digitação com lições programadas ou criadas pelo professor (precisa de cadastro). <br>
  * <a href="https://mouse-practice.com/">Mouse House</a>: Manuseio de mouse com jogos para crianças pequenas.<br>
  * <a href="https://minimouse.us/VineTime.htm">MiniMouse</a>: Jogo para manuseio de mouse.<br>
  * <a href="https://www.roomrecess.com/games/DragonDrop/play.html">Dragon Drop</a>: Manuseio de mouse.<br>
<br>

* Jogos educativos de outros temas:<br>
  * <a href="https://beinternetawesome.withgoogle.com/pt-br_br/interland/">Interland</a>: Ensina cidadania digital e combate ao ciberbullyng. Criado pela própria Google. Indicado a partir dos 7 anos (crianças alfabetizadas).<br>
  * <a href="https://www.escolagames.com.br/jogos-educativos/criancas-pequenas">Escola Games</a>: Plataforma de jogos para crianças a partir de 4 anos de termas educativos diversos. Indicado para crianças em fase de alfabetização.<br>
